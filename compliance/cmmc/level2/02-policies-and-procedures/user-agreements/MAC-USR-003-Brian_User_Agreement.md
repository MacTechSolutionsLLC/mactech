## Acknowledgment Statements

By signing below, I acknowledge that I have read and understand the following:

### 1. Federal Contract Information (FCI) Protection

☐ **I understand my responsibility to protect Federal Contract Information (FCI)** as defined by FAR 52.204-21. I will:

- Treat all FCI with appropriate care and confidentiality
- Access FCI only for authorized business purposes
- Not disclose FCI to unauthorized individuals
- Report any suspected FCI breaches or unauthorized access
- Follow all organizational policies and procedures related to FCI handling

### 2. Prohibition on Controlled Unclassified Information (CUI)

☐ **I understand that Controlled Unclassified Information (CUI) must be properly marked and password protected.** I will:

- Mark files containing CUI during upload
- Use password protection when accessing CUI files
- Follow CUI handling procedures as documented
- Report any suspected CUI breaches or unauthorized access
- Understand that CUI files are stored separately and require password protection

### 3. System Access and Security

☐ **I understand that system access is granted based on business need and least privilege principles.** I will:

- Use my account only for authorized purposes
- Not share my account credentials with anyone
- Not attempt to access data or functions beyond my authorized scope
- Report any suspected security incidents or unauthorized access attempts
- Change my password if I suspect it has been compromised

### 4. Data Handling

☐ **I understand that FCI is stored only in the cloud database and not on local devices.** I will:

- Not download FCI to local devices unless explicitly authorized
- Not store FCI on removable media
- Not print FCI unless authorized and necessary for business purposes
- Securely dispose of any printed FCI when no longer needed

### 5. Ongoing Compliance Requirements

☐ **I understand that I have ongoing compliance responsibilities** as documented in the Ongoing Stakeholder Requirements document (`MAC-POL-217_Ongoing_Stakeholder_Requirements.md`). I will:

- Comply with password management requirements
- Participate in access reviews and least privilege audits
- Report security incidents immediately
- Maintain physical access logs (if applicable to my role)
- Maintain endpoint inventory (for endpoints I use to access the system)
- Complete annual re-acknowledgement of these requirements
- Notify administrators of role changes or access need changes
- **Fulfill administrative responsibilities (ADMIN role):**
  - Review user access periodically
  - Conduct least privilege audits
  - Revoke access when triggers occur (within 24 hours)
  - Maintain account lifecycle documentation
  - Review and approve access requests
  - Monitor system security events
  - Review audit logs periodically
  - Respond to security incidents
  - Maintain system documentation
  - Coordinate with Security Contact on security matters
  - Maintain physical access logs
  - Maintain endpoint inventory
  - Export audit logs for evidence (as needed)
  - Maintain user acknowledgement forms
  - Document access reviews and audits

**I acknowledge that I have read and understand:** `MAC-POL-217_Ongoing_Stakeholder_Requirements.md`

### 6. Consequences of Violation

☐ **I understand that violation of these requirements may result in:**

- Immediate revocation of system access
- Disciplinary action up to and including termination
- Legal consequences as applicable
- Reporting to the Security Contact (security@mactechsolutions.com) and, if contractually required, to the contracting officer or customer
