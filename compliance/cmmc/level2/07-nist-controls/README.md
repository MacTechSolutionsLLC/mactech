# NIST SP 800-171 Controls Assessment Files

**Purpose:** This folder contains comprehensive assessment files for all 110 NIST SP 800-171 Rev. 2 controls required for CMMC Level 2 compliance.

**Structure:**
- One file per control: `NIST-3.X.Y_Control_Title.md`
- Total files: 110 (one for each NIST SP 800-171 control)
- Index file: `00-INDEX.md` (provides navigation to all controls)

**File Contents:**
Each control file includes:
- Control ID and full requirement text
- Implementation status and justification
- Policy and procedure references
- Implementation evidence (code, configuration, operational)
- Evidence document references
- Testing and verification details
- SSP section references
- Related controls
- Assessment notes
- POA&M references (if applicable)

**Data Sources:**
- Primary: System Control Traceability Matrix (SCTM)
- Secondary: Existing MAC-RPT evidence files
- Additional: System Security Plan, POA&M, Inherited Controls documentation

**Usage:**
These files serve as comprehensive assessment packages for each control, enabling assessors to:
- Understand the control requirement
- Review implementation status
- Access all related evidence
- Verify testing and verification activities
- Track POA&M items (if applicable)
