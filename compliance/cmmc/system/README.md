# System Security Documentation - CMMC Level 2

**Purpose:** General system security documentation (not FCI-specific)

**Scope:** System architecture, security controls, policies, and procedures that apply to the entire system regardless of data type (FCI, CUI, or general system data).

## Directory Structure

- `00-cover-memo/` - Executive attestations and cover memos
- `01-system-scope/` - System Security Plan (SSP), System Description, System Architecture
- `02-policies-and-procedures/` - General security policies and procedures
- `03-control-responsibility/` - Inherited controls documentation
- `04-self-assessment/` - Self-assessments, SCTM, POA&M
- `05-evidence/` - General security control evidence
- `06-supporting-documents/` - Supporting documentation
