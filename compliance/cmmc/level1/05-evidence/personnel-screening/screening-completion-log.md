# Personnel Screening Completion Log - CMMC Level 2

**Document Version:** 1.0  
**Date:** 2026-01-23  
**Classification:** Internal Use  
**Compliance Framework:** CMMC 2.0 Level 2 (Advanced)  
**Reference:** NIST SP 800-171 Rev. 2, Section 3.9.1

**Control:** 3.9.1 - Screen individuals prior to authorizing access to organizational systems containing CUI

---

## 1. Purpose

This document tracks personnel screening completion for all individuals with access to systems containing CUI.

---

## 2. Screening Requirements

**Screening Required For:**
- All personnel requiring access to systems containing CUI
- All personnel with system access credentials
- Screening conducted before access authorization

**Screening Types:**
- Employment verification
- Reference check
- Background check (as applicable)
- Security clearance (if required by contract)
