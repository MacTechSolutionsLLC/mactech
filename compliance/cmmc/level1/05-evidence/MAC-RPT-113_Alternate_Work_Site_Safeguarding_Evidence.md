## 3. Alternate Work Site Safeguarding

### 3.1 Work Area Security

**Safeguarding Measures:**
- Work area in private, controlled space
- Work area secured when unattended
- Screens positioned to prevent unauthorized viewing
- Sensitive information not left visible
- Access restricted to authorized personnel

---

### 3.2 Device Security

**Device Requirements:**
- Devices password-protected
- Screen lock enabled with automatic timeout
- Operating system kept up to date
- Antivirus/antimalware installed (if applicable)
- Devices kept secure when unattended

---

### 3.3 Network Security

**Network Controls:**
- All access via HTTPS/TLS
- Public Wi-Fi avoided or used with VPN
- Home network secured (WPA2/WPA3)
- Network encryption enforced

---

### 3.4 CUI Handling

**CUI Protection:**
- CUI accessed only from controlled devices
- CUI not stored on local devices
- CUI accessed via secure network connections
- CUI handling procedures followed
- CUI spillage reporting procedures
